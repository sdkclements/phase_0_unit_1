Why it's Important?


  • You can bounce ideas off each others head
  • You can motivate one another
  • Learn from each other's past experience
  • None of us are as dumb as all of us =)
  • Useful tool in the business world

What does it mean to be a Driver?


  • Commiting the Code to the page
  • Interpreting what Navigator is saying
  • Making suggestions

What does it mean to be a Navigator?


  • Outlining what the code should do
  • Directing what kind of code should be used
  • Where the code should be used
  • Making suggestions

Symbiotic Relationship

Navigator and Driver should be able to understand each other's positions and be able to switch positions when necessary.
